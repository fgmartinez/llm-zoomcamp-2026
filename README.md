# llm-zoomcamp-2026

This project is configured for the LLM Zoomcamp homework environment.

## Setup

1. Create a virtual environment:
   ```powershell
   python -m venv .venv
   .\.venv\Scripts\Activate.ps1
   ```
2. Install dependencies:
   ```powershell
   python -m pip install -U pip
   python -m pip install -r requirements.txt
   ```

## Notes

- Add your OpenAI API key to `.env` based on `.env.example`.
- Do not commit `.env` to git.
